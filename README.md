## 🔬 Research Contributions

### 🎯 Core Research Achievements

**🔬 Machine Unlearning Privacy Security System**

- **📊 Unified Evaluation Framework**: Developed the first five-dimensional assessment protocol for sample-level machine unlearning
- **⚔️ Attack Method Integration**: Systematically integrated LiRA, REA, RULI, UnlearningLeaks for comprehensive evaluation
- **🛡️ Novel Defense Strategy**: Proposed MISE (Membership Identity Suppression for Erasure) with theoretical guarantees
- **🌐 Open-Source Tool**: Created reproducible research framework for the community

### 📂 Featured Project: MIA-benchmark ⭐

[![MIA-benchmark](https://github-readme-stats.vercel.app/api/pin/?username=keke2510&repo=MIA-benchmark&theme=radical&hide_border=true&bg_color=0D1117&title_color=60A5FA&icon_color=F59E0B&text_color=FFFFFF)](https://github.com/keke2510/MIA-benchmark)

**🏆 Significance:** This work addresses the critical gap in unified evaluation protocols for machine unlearning privacy risks.

**🎯 Key Innovations:**
- **First unified framework** for sample-level machine unlearning MIA evaluation
- **Five-dimensional assessment** covering attack success rate, robustness, efficiency, scalability, and defense
- **Novel theoretical foundation** based on membership indistinguishability constraints
- **Practical defense method** with proven effectiveness in real-world scenarios

**📊 Research Impact:**
- Provides standardized benchmarks for future research
- Enables reproducible privacy risk assessment
- Offers practical defense solutions for ML systems
